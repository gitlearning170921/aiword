# 打包说明

## 方法一：使用打包脚本（推荐）

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **运行打包脚本**
   ```bash
   python build_exe.py
   ```

3. **找到生成的 exe 文件**
   - 打包完成后，exe 文件位于 `dist/文档生成工具.exe`
   - 可以直接双击运行

## 方法二：手动使用 PyInstaller

1. **安装 PyInstaller**
   ```bash
   pip install pyinstaller
   ```

2. **打包命令**
   ```bash
   pyinstaller --name=文档生成工具 --onefile --windowed aiword.py
   ```

3. **参数说明**
   - `--name=文档生成工具`: 生成的 exe 文件名
   - `--onefile`: 打包成单个 exe 文件（不包含依赖文件夹）
   - `--windowed`: 不显示控制台窗口（适合 GUI 程序）

## 方法三：使用 spec 文件（高级）

如果需要更精细的控制，可以先生成 spec 文件：

```bash
pyinstaller --name=文档生成工具 --onefile --windowed aiword.py
```

然后编辑生成的 `文档生成工具.spec` 文件，再次打包：

```bash
pyinstaller 文档生成工具.spec
```

## 注意事项

1. **首次打包较慢**：PyInstaller 需要分析所有依赖，首次打包可能需要几分钟
2. **杀毒软件误报**：某些杀毒软件可能会误报，这是正常现象，可以添加信任
3. **文件大小**：打包后的 exe 文件可能较大（几十到上百 MB），因为包含了 Python 解释器和所有依赖
4. **兼容性**：在 Windows 10/11 上打包的 exe 通常可以在其他 Windows 系统上运行

## 测试

打包完成后，建议：
1. 在打包的电脑上测试 exe 文件
2. 在没有安装 Python 的电脑上测试（确保所有依赖都打包进去了）
